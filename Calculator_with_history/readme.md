# 🧮 Simple Calculator with History (Python)

A beginner-friendly Python project that performs basic arithmetic operations and saves calculation history to a file.

## 🚀 Features

* Supports basic operations: `+`, `-`, `*`, `/`
* Accepts flexible input (e.g., `8+8`, `8 + 8`, `10/2`)
* Saves all calculations to a history file
* View past calculations using `history`
* Clear saved history using `clear`
* Continuous loop until user exits

## 🛠️ How It Works

* The user enters a calculation or command
* The program processes the input and performs the operation
* Each result is saved in `history.txt`
* History is displayed in reverse order (latest first)

## ▶️ How to Run

1. Make sure Python is installed
2. Run the script:

```bash
python main.py
```

## 📌 Example Usage

Enter calculation (+ - * /) or command(type history, clear or exit): 8+8
Result: 16

8 + 8 = 16


## 🧠 Commands

* `history` → View past calculations
* `clear` → Delete all history
* `exit` → Close the program

## 📂 File Used

* history.txt → Stores all calculations
