# 📰 Fake News Headline Generator

A fun Python mini-project that generates random, humorous “breaking news” headlines using predefined subjects, actions, and places.

## 🚀 Features

* Generates unique headlines every time
* Uses Python’s `random` module
* Simple and beginner-friendly logic
* Interactive loop (keeps generating until user exits)

## 🛠️ How It Works

The program randomly selects:

* a **subject** (e.g., celebrity, person, or character)
* an **action** (e.g., eats, launches, argues about)
* a **place or thing** (e.g., at Taj Mahal, during IPL match)

It then combines them into a headline like:

> BREAKING NEWS: A Delhi Chaiwala secretly builds a plate of biryani at Dubai Marina

## ▶️ How to Run

1. Make sure Python is installed
2. Clone this repository or download the file
3. Run the script:

python main.py

## 📌 Example Output

BREAKING NEWS: A Bangalore Software Engineer dances with a group of cats inside a college library
